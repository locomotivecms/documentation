# Locomotive CMS documentation

[http://doc.locomotivecms.com]([http://doc.locomotivecms.com)


## [en] English version, french version follow 
- - -
This repository is for hosting the documentation for Locomotive CMS.

You can help us to have a better documentation with this method :

* Fork this repositiry
* Clone it on your work station
* Install [Locomotive Editor](http://www.locomotivecms.com/modules/locomotive-editor)
* Modify the pages with your favorit editor
* Visualize your modification with Locomotive Editor
* Submit a Pull Request on github

Thank you for your help


## [fr] Version française
- - -
Ce dépôt est pour héberger la documentation de Locomotive CMS.

Vous pouvez nous aider afin d'obtenir une meilleure documentation avec cette méthode :

* Faites un “fork” de ce dépôt
* Clonez-le sur votre poste de travail
* Installez [Locomotive Editor](http://www.locomotivecms.com/modules/locomotive-editor)
* Modifiez les pages de cette documentation avec votre éditeur favori
* Visualisez vos modifications avec Locomotive Editor
* Soumettez un “pull request” sur github

Merci d'avance de votre aide
